## 🧪 Test the Application

### 1. Setup Freighter Wallet
```
1. Install Freighter extension: https://www.freighter.app/
2. Create or import wallet
3. Switch to Testnet in settings
4. Fund wallet: https://friendbot.stellar.org
```

### 2. As Landlord
```
1. Visit: https://rentledger-pro.vercel.app
2. Connect Freighter wallet
3. Go to Dashboard → Landlord tab
4. Create a property
5. Copy the payment link
```

### 3. As Tenant
```
1. Open the payment link
2. Connect Freighter wallet
3. Review property details
4. Pay rent
5. Get instant blockchain receipt
```
